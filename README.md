# Notiway Community

Welcome to the Notiway community hub! This repository is the central place for bug reports, feature requests, and discussions related to Notiway.

## How to participate

- **Bug reports** — [Open a bug report](https://github.com/Notiway/notiway-community/issues/new?template=bug_report.md)
- **Feature requests** — [Request a feature](https://github.com/Notiway/notiway-community/issues/new?template=feature_request.md)
- **Discussions** — [Join the conversation](https://github.com/Notiway/notiway-community/discussions)

## About Notiway

Notiway is a lightweight gateway for in-app notifications. Built on SignalR with a pluggable, cloud-agnostic architecture and open-source plugins.

- [Documentation](https://notiway.dev)
- [Plugin repositories](https://github.com/orgs/Notiway/repositories)

## Security

If you discover a security vulnerability, please use [GitHub's private vulnerability reporting](https://github.com/Notiway/notiway-community/security/advisories/new) instead of opening a public issue.
